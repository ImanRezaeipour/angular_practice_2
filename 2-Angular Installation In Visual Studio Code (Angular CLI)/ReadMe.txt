Using Angular In Visual Studio Code

1- Install Node.js ---> nodejs.org

2- Install "Visual Studio Code" ---> code.visualstudio.com

3- Check Npm Installation ---> [Npm --version]

4- Scafold Angular Structure [npm install -g "@angular/cli"]

5- Create Angular App --> [ng new "ApplicationName"]

6- Running Web Server --> [ng serve]

7- Debugging Client Side Code with "Debugger for Chrome"
